# EcoEnchants_zh_CN

EcoEnchants汉化

汉化版本基于`EcoEnchants v12.19.3`

附魔名称手工翻译，我将根据自己的使用习惯，修改为我认为更贴合附魔效果描述的中文名称，更适合中国宝宝体质

翻译原则：

​	1.直译与意译结合

​	2.用词向原版附魔名称靠拢

​	3.通俗易懂

如果觉得我的翻译不准确，或者有更合适的翻译，欢迎贡献您的智慧，迫切希望您的参与！
