# EcoEnchants_zh_CN



EcoEnchants汉化

汉化版本基于`EcoEnchants v12.19.3`

附魔名称手工翻译，我将根据自己的使用习惯，修改为我认为更贴合附魔效果描述的中文名称，更适合中国宝宝体制

翻译原则：

​	1.贴近原版附魔名称，不标新立异

​	2.名称应当准确符合描述，通俗易懂

如果您觉得我的翻译不准确，或者有更合适的名称，欢迎贡献您的智慧
